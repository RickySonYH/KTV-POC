실시간 자막 표시 규칙을 설정합니다. 변경사항은 즉시 프론트엔드에 적용됩니다.

화면에 표시할 자막 줄 수 (1~5)

한 줄에 표시할 최대 글자 수 (10~50)

묵음이 지속되면 자막 페이드아웃 (1000~10000ms)

자막 표시 전 대기 시간 (0=실시간)

자막이 최소한 표시되어야 하는 시간

마침표(.), 물음표(?), 느낌표(!) 후 줄바꿈

자막 표시 방식

  • 아랫줄부터 채워지고, 꽉 차면 윗줄로 이동
  • 문장이 끝나면 (마침표 등) 즉시 줄바꿈
  • 묵음이 지속되면 페이드아웃으로 정리
  • 후처리 규칙 변경 시 이미 표시된 자막도 교체 가능
